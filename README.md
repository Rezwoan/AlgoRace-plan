# AlgoRace — Project Plan

> Private planning repository for [AlgoRace](https://github.com/Rezwoan/AlgoRace)

This repo contains the detailed 14-day development plan, specs, and checklists for the AlgoRace project — a maze race game with pathfinding visualization built in Python + Pygame.

## Structure

- `docs/specs/` — Constants, file structure, state machine, data model
- `docs/week-1/` — Days 1–7: Core engine & gameplay
- `docs/week-2/` — Days 8–14: Persistence, CRUD, analytics, polish
- `docs/checklists/` — MVP and final checklists
